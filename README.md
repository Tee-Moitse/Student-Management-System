# Student-Management-System

# Description
This project is a management system that keeps record of students and all their records at their institution. The purpose of this project is to create a convenient way of managing student records using one of Python's data structures that holds information about each student in the institution.

# Features
The features of this project:
- Add student
- View student
- Search student
- Delete student
- Update student

# Tech Stack
- Python
